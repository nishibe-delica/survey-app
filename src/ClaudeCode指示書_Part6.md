## AdminWhitespaceView.tsx（④ ホワイトスペース全体ビュー）

全お客様のホワイトスペース状況を俯瞰できる画面。

```typescript
// 切り替えボタン（上部）
// 「👤 お客様別」←→「👥 担当営業別」
// 選択中：ネイビー背景・白文字
// 未選択：白背景・グレー文字

// 【お客様別ビュー】
// 各お客様を横1行のカードで表示
// 左：会社名・担当営業名

//   🟢緑：スイテック
//   🟠オレンジ：競合
//   🟡黄：検討中
//   ⚪グレー：未導入
//   ➖：情報なし

// 【担当営業別ビュー】
// 担当営業ごとにセクションを分けて表示
// 各セクション：
//   営業名（大）/ 担当件数
//   配下のお客様カード一覧（お客様別ビューと同じ形式）
// 営業名でアコーディオン開閉可能

// Supabaseから全回答データを取得
// ホワイトスペース判定は既存のutils/whitespace.tsを使用
```

## Supabaseデータ取得の注意事項

```typescript
// 既存のSupabaseクライアント設定を使用すること
// src/lib/supabase.ts または src/utils/supabase.ts を確認して使用

// データ取得例（テーブル名は既存のものを確認）
const { data, error } = await supabase
  .from('既存のテーブル名')  // ← 必ず既存テーブル名を確認してから使用
  .select('*')
  .order('created_at', { ascending: false });

// エラー時の処理を必ず実装すること
// ローディング中はスピナーを表示すること
// データが0件の場合は「まだ回答がありません」と表示すること
```
